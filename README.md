## 설명

### [실전 외주 웹사이트 퍼블리싱]

- DoctorPage 웹사이트 외주 퍼블리싱입니다.

```
- 반응형 사이트입니다.
- 로그인, 회원가입, 장바구니를 제외한 메뉴에 있는 모든 페이지를 퍼블리싱 하였습니다.
- 기능들은 제이쿼리를 사용하여 구현하였습니다.
- 슬라이더는 slick을 이용하였습니다.
- 레이아웃은 css display flex를 이용하였습니다.
```
